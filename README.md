# Multi-Agent Interview Trainer

Мультиагентная система для проведения тренировочных технических собеседований по IT-вакансиям (Backend, Analyst, Data, QA и др.).

Проект реализует **Agentic Workflow**, а не просто чат-бот:
логика интервью распределена между несколькими специализированными агентами, а LLM используется как инструмент анализа, а не как единый управляющий промпт.

---

## Возможности

- Проведение интерактивного технического интервью (Вопрос → Ответ)
- Поддержка грейдов: Junior / Middle / Senior
- Адаптивная сложность вопросов
- Устойчивость к:
  - off-topic ответам (уход в разговоры о погоде, хобби и т.д.)
  - уверенным ложным утверждениям (hallucinations)
  - попыткам перехватить инициативу (role reversal)
- Скрытая рефлексия агентов (внутренние рассуждения логируются, но не показываются кандидату)
- Генерация структурированного финального фидбэка
- Логирование интервью в JSON-формате

---

## Архитектура агентов

Система состоит из нескольких ролей:

### Interviewer
- Ведёт диалог с кандидатом
- Формулирует вопросы
- Вежливо возвращает беседу к теме интервью

### Observer
- Анализирует ответы кандидата «за кулисами»
- Классифицирует ответы (`STRONG / WEAK / OFFTOPIC / HALLUCINATION / REFUSAL`)
- Управляет сложностью вопросов
- Передаёт инструкции Interviewer’у

### Verifier (LLM-based)
- Использует LLM (Mistral) для универсальной проверки:
  - релевантности ответа вопросу
  - наличия ложных или абсурдных утверждений
  - попыток смены роли или отказа отвечать
- Повышает устойчивость системы к непредсказуемым сценариям

---

## Используемые модели

- **mistral-small-latest** — основной LLM для:
  - анализа ответов
  - проверки фактов (Verifier)
  - генерации follow-up вопросов
- DummyLLM — fallback для локального запуска без API-ключа
